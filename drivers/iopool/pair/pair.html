Authentication / Authentification

EN: To authenticate with the API, you need an API key linked to your iopool account. You can find your API key at the bottom of the Settings page in the iopool app.

FR: Pour vous authentifier auprès de l’API, vous devez obtenir une clé API liée à votre compte iopool. Vous trouverez votre clé API en bas de la page Paramètres de l’application iopool.
